# LP多言語化（日・英）実装計画 (2026-03-13)

「椎の間」LPを日本語と英語の2言語に対応させ、ユーザーが任意に切り替えられる機能を実装します。

## Proposed Changes

### 1. 言語切替機能の実装
- **ヘッダーへのスイッチャー追加**: `index.html` のヘッダー右側に「JP / EN」の切替ボタンを設置します。
- **表示切り替えロジック**: 
  - `body` タグに `lang-ja` または `lang-en` クラスを付与します。
  - スタイルシートで、選択されていない言語の要素を `display: none` にします。
  - 選択状態は `localStorage` に保存し、再訪時も維持します。

### 2. コンテンツの bilingual 化
- 各テキスト要素を、日本語用と英語用の2つの要素（またはデータ属性）に分けます。
- 残っていたメタ情報やフッターリンク、ロゴのalt属性等も全てバイリンガル化。

### 3. スタイリングの調整
- 英語表示時にフォントサイズや行間が崩れないよう、言語に応じたCSS微調整を行います。
- スイッチャー自体のデザインも、侘寂のイメージを崩さないミニマルなものにします。

## Verification Plan

### Automated Tests
- ブラウザツールを使用し、切替ボタンで全ての文言が切り替わることを確認。
- `localStorage` に保存され、リロード後も選択言語が維持されることを確認。

### Manual Verification
- 英語表示時のレイアウト崩れ（テキストの溢れなど）がないか実機で確認。
- 動的なページタイトルの切り替えを確認。
